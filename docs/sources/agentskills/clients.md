# Client Showcase

> Agent products that support the Agent Skills format.

Agent Skills are supported by leading AI development tools including:

- **Junie** - LLM-agnostic coding agent built on the IntelliJ Platform
- **Gemini CLI** - Open-source AI agent bringing Gemini to your terminal
- **Autohand Code CLI** - Autonomous LLM-powered coding agent
- **OpenCode** - Open source agent for terminal, IDE, or desktop
- **OpenHands** - Open platform for cloud coding agents
- **Mux** - Run parallel coding agents in your browser
- **Cursor** - AI editor and coding agent
- **Amp** - Frontier coding agent for leading models
- **Letta** - Platform for building stateful agents
- **Firebender** - Android-native coding agent
- **Goose** - Open source extensible AI agent
- **GitHub Copilot** - AI pair programmer
- **VS Code** - Code editor with Copilot
- **Claude Code** - Agentic coding tool
- **Claude** - Anthropic's AI assistant
- **OpenAI Codex** - OpenAI's coding agent
